# Tail TUI

## Overview

The tail TUI is the terminal UI component responsible for rendering log lines. It presents each line alongside a gutter and arranges the overall on-screen layout. The component's product-level behavior and interaction model are documented in `docs/product-spec/tail-ux.md`, and its system-level architecture and implementation details are documented in `docs/product-spec/tail-system.md`. <!-- [^26c90-9d987] -->
