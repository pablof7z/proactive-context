# pc agents Display

## Display

The `pc agents` output is colorized. The header uses a bold white label and a bold cyan repo path. The `●` bullet is green for active, dimmed for done, and yellow for expired. The session ID is dimmed. Status color matches the bullet. The `[branch, age]` is cyan. The worktree path is dimmed. Footer hints are dimmed. <!-- [^28cf7-f2508] -->
