# pc configure Models Pane

## Provider Tab Bar

The pc configure models pane uses a provider tab bar with tabs for All / Claude CLI / OpenRouter / Ollama, filtering the model list to the selected provider. Each tab shows a live count of models, and the active tab is highlighted cyan-on-black. The `[` and `]` keys cycle through provider tabs. The help bar mentions `[/]` for tabs. <!-- [^d8d7d-4f3e3] -->
