# --in flag (pc capture --in)

Routes capture into the debounced path — only Some vs None matters; the number value is ignored (vestigial). It functions as a boolean 'debounce this' flag that happens to carry a misleading number. After the refactor, bare --in reads config and --in N provides an override.
