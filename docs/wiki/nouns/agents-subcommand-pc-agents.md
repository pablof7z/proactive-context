# Agents subcommand (`pc agents`)

Shows the cross-agent standup board for the current repo: every concurrent Claude Code agent's branch, age, status, and distilled intent — an on-demand snapshot (the awareness hooks otherwise only surface ephemeral deltas after tool calls).
