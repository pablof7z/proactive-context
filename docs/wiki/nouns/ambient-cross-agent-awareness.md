# ambient cross-agent awareness

A per-repo 'standup board': concurrently-running Claude Code agents across worktrees of the same repo distill a one-line statement of what they are working on into a shared SQLite agents.db, and PostToolUse hooks surface only deltas (new peer, changed intent, finished peer) as system-reminders.
