# librarian (compile model role)

The inject compile model's role: a librarian, not an answerer. Given a search query (the prompt), it gathers relevant wiki sections and returns filepath:line-range verbatim slices + each wiki's relative last-updated date — it does not write prose or attempt to answer the user's question.
