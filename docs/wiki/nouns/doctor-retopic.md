# doctor --retopic

A one-shot global LLM taxonomy pass that re-stamps the topic field across the existing flat wiki to collapse narrow/singleton topics into broad ones; defaults to dry-run (read-only), rewrites the topic field only (zero file moves, zero storage changes), and regroups _index.md for free.
