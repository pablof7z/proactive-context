# COMPILE (compile stage)

Originally defined as 'which lines within those documents are relevant?' — a line-range selector returning JSON {slug, start, end} that Rust slices verbatim. Redesigned mid-session to a synthesizer: produces a dense synthesized briefing in its own words with mandatory inline (path:line) citations on every claim, no longer returning line ranges.
