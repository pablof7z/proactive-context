# pc oracle

A proposed subcommand that loads all ground truths (wiki guides + eventually user-origin claims) into a single large-context model call, with modes: `<question>` for full-wiki Q&A, `--interactive` for REPL, `--recompile` for global consistency pass, and `--gaps` for missing-topic detection
