# RECONCILE

The stage that groups all claims by their target guide and processes one guide at a time, showing the model the full current guide body plus all claims routed to it this session, so the model sees the contradiction it would create and decides per claim: add / revise / remove / propose-new. Explicitly sequential, not parallel.
